An example of <ll-date-input>

When date is set via picker...

  • the input's value is set to the "M/DD/YYY" format.
  • the data-data attribute is set to the "YYYY-MM-DD" format.
  • the data-data attribute is two-way bound, so you can bind it to a property of the Polymer element that contains this input; eg:
    <input is="ll-date-input" data-data={{property.checkInDate}}/>;
